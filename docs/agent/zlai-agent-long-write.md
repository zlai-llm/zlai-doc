# ZLAI Agent Write

> 什么是 `Agent Write` ？

简而言之，`Agent Write`就是通过`Agent`的方式让模型生成更为准确，更为详实的回答。

**在上面的架构中，先让大模型生成了文章的大纲，然后再让大模型依据大纲生成各个自然段的内容。**

## Agent Write

### Long Write Agent Plan

> 生成文章大纲，示例代码

```python
from zlai.llms import Zhipu, GLM4FlashGenerateConfig
from zlai.agent.long_write import *

llm = Zhipu(generate_config=GLM4FlashGenerateConfig())

task = LongWriteAgentPlan(llm=llm, verbose=True)
completion = task("杭州旅行指南")
print(completion.content)
```

### Long Write Agent Write

> 依据文章大纲生成文章，示例代码

```python
from zlai.llms import Zhipu, GLM4FlashGenerateConfig
from zlai.agent.long_write import *

llm = Zhipu(generate_config=GLM4FlashGenerateConfig())

task_plan = LongWriteAgentPlan(llm=llm, verbose=True)
completion_plan = task_plan("杭州旅行指南")
print(completion.content)

task = LongWriteAgentWrite(llm=llm, verbose=True)
completion = task(query=completion_plan)
print(completion.content)
```

### LongWriteAgent

> `LongWriteAgent`是`Plan/Write`的`TaskSequence`封装，可以顺序执行文章大纲的生成与正文编写。

> 示例代码

```python
from zlai.llms import Zhipu, GLM4FlashGenerateConfig
from zlai.agent.long_write import *

llm = Zhipu(generate_config=GLM4FlashGenerateConfig())

task = LongWriteAgent(llm=llm, verbose=True)
completion = task(query="杭州旅行指南")
print(completion.content)
```

**下面展示两个模型版本的示例。**

- `GLM4LongWriter`: 需要按照该部分的开源模型[部署](llm/zlai-llm-openai-api)进行本地模型接口部署。
- `GLM4Flash`: 智谱提供的大模型接口。
